TAGS NO HTML5 SÃO PURAMENTE SEMÂNTICA!

No html4, era possível utilizar tags que não eram puramente semânticas, como "center", "marquee", "bgcolor" como attribute do body, etc... todas elas são passíveis de desuso, pois no html5 não se usam mais. Esse tipo de efeito é desejado apenas pelo CSS hoje em dia, que cuida dos estilos da página.

Nesta frase, tenho um termo em destaque usando a tag "strong" (semântica).

Itálico / Ênfase

Nesta fraase, temos um termo em itálico usando a tag "i" (não semântica)

Nesta frase, temos um termo em ênfase usando a tag "em" (semântica).

Texto marcado

Podemos também criar um texto marcado usando a tag "mark". OBS: a cor da marcação foi feita com CSS!

E no outro parágrafo, temos outro texto marcado no final. OBS: Note que, neste caso, nós alteramos a cor padrão ao criar um style no Header!

Texto grande e pequeno

Estamos crfianod um texto grande e um texto pequeno neste parágrafo com as tags "big" (desuso) e "small".

Texto deletado

Podemos marcar um texto como excluído (aparece tracejado, mas tem semântica) para indicar que ele deve ser lido, mas não considerado. Tag: "del".

Texto inserido

Podemos marcar um texto como inserido (aparece sublinhado, mas tem semântica) para dar uma ênfase e indicar que ele foi adicionado depois. Tag: "ins".

Existe também o sublinhado de fato, com a tag "u" (não semântica) com a tag "u"

Texto sobrescrito

Para inserir coisas do tipo x20+3. Tag: "sup"

Texto subscrito

Para inserir coisas do tipo H2O. Tag: "sub"
